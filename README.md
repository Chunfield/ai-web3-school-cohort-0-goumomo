# AI x Web3 School Learning Repo

Personal learning journal and proof-of-work for AI x Web3 School.

## Links

- Handbook: https://aiweb3.school/zh/handbook/
- WCB Program: https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning: https://web3career.build/zh/programs/AI-Web3-School#tab=learning

## Privacy Reminder

This repository is public. Do not commit API keys, wallet mnemonics, private keys, passwords, internal meeting links, unpublished contact details, or any other sensitive data.

## Structure

- `profile.md`: learner profile and operating rules
- `learning-plan.md`: current learning plan and milestones
- `daily/`: daily notes and check-in drafts
- `tasks/`: task breakdowns and work logs
- `experiments/`: code, demos, and experiment notes
- `handbook-feedback/`: feedback records for the Handbook
- `hackathon/`: project ideas, build notes, and demo prep
- `submissions/`: links and records for final submissions
- `templates/`: reusable note templates

## Working Rules

1. Learn in public, but keep secrets local.
2. Prefer small daily proof-of-work over long passive reading.
3. Convert blockers into reusable notes, experiments, or feedback records.
4. Confirm content before any write action to external platforms.

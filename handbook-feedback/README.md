# Handbook Feedback Workflow

Use this folder to record unclear concepts, broken links, outdated content, structure suggestions, and typo fixes for the AI x Web3 School Handbook.

## Record Format

Each feedback entry should include:

- Handbook page link
- Problem summary
- Why it blocks learning or causes confusion
- Suggested fix
- Evidence or source

## Suggested Flow

1. Capture the issue locally first.
2. Make the description concrete and reproducible.
3. Keep one issue per file when possible.
4. Share or submit only after the wording is clear.

## File Naming

Use: `YYYY-MM-DD-short-topic.md`
